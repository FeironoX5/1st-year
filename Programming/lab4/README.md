# Лабораторная №4
## Задание

В которой нужно доработать лабораторную №3,
реализовав 2 собственных класса исключений
checked и unchecked, а также их обработку исключений этих классов.
Также в программу требуется добавить:
- [ ] локальные классы
- [ ] анонимные классы
- [ ] вложенные классы
